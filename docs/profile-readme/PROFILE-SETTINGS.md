# Profile settings — bio, avatar & fields

These are the fields that sit **beside** your README on your profile page (Edit profile → the panel on the left). Getting them consistent with the README is what makes the whole thing read as _authored_, not assembled.

## Avatar

Use [`avatar.png`](./avatar.png) (460×460, rendered from [`avatar.svg`](./avatar.svg)) — a terracotta **AT** monogram with the firefly spark, matching your studio brand.

- Upload at **Settings → Profile → Profile picture**.
- To recolor or tweak, edit `avatar.svg` and re-export to PNG.

## Bio (the 160-character field)

- **Pin the right repos** (Settings → Customize your pins) so your pinned grid echoes the "Selected work" table in the README — same story, twice.
